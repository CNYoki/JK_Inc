JK_Inc
====

浙江省金控微信H5创业公司调查反馈，邀请码（限用一次）



### 基于

nodejs、mongoose、ejs、nodemailer、grunt

同类产品 ： [hi-blog][2]

Author : [HighSea][1] 

### 安装

	git clone https://github.com/highsea/JK_Inc.git
	cd JK_Inc && npm install
	node app
